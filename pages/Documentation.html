Subdora 0.2.2

Subdora is a Python obfuscation tool designed to transform your code into a highly secure, non-readable format. Built to protect source code from unauthorized inspection and tampering, it’s backward-compatible with version 0.2.1, making it a flexible solution for securing complex Python algorithms.

Obfuscating a python file

main.py

print('hello')

using CLI

~$ subdora --obfuscate main.py

This command create main.myst file

main.py file can be deleted

in order to run .myst file

~$ subdora --run main.myst
~$ hello

using python file

import Subdora
Subdora.subdora_encode_file("main.py")

Additional arguments

~$ subdora --obfuscate main.py --img image_path.jpg --itr 5 --time 10m

Above example shows how we obfuscate main.py file into an image with iteration counter of 5 and file is valid only for 10 minutes

Subdora only supports .png and .jpg files

--time argument support minutes(m) and hours(h)
